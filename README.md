# Book Library

## Description
This project is a web application built with Angular and Django REST Framework. Users can browse books, view details, and leave reviews.

## Main features
- home page with polished modern UI
- books catalog with search, category filter and sorting
- detailed book page
- user registration and login
- user profile
- logout
- reviews with duplicate-review protection
- expanded demo catalog: 10 categories and 20 books

## Tech stack
- Angular
- TypeScript
- Django
- Django REST Framework
- SQLite
